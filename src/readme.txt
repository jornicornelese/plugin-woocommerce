== Installation ==

= General instructions =

1. Upload the plugin files to the `/wp-content/plugins/biller-business-invoice` directory,
2. Activate the plugin through the 'Plugins' screen in WordPress or install the plugin through the WordPress plugins screen directly (recommended).

= Checking the installation =

Your customers should be able to select Biller payment method in the checkoutpage.
